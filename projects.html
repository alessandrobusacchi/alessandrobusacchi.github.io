Speech-Driven 3D Facial Animation with ARKit Blendshape Representation
MSc Thesis · 2025–2026
Utrecht University · Computer Vision / Graphics / AI

The goal of this project is to bridge academic speech-driven facial animation models with production-ready animation pipelines by leveraging a semantic and artist-friendly ARKit blendshape representation for control and integration in real-time systems.

The work focuses on building a multimodal pipeline combining visual and audio data to generate ARKit blendshape representations, enabling controllable and expressive digital humans.

  • Built a multimodal pipeline combining audio and visual data for facial animation generation
  • Created a high-quality ARKit blendshape dataset from monocular video using MediaPipe
  • Adapted and trained speech-driven models to predict ARKit blendshape coefficients
  • Explored architectural improvements to increase expressiveness and fidelity
  • Developed a production-ready pipeline for integration into real animation workflows
  • Conducted user studies and qualitative/quantitative evaluation of realism and usability
